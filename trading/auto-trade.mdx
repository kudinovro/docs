Tap **Auto Trade** from the main menu to create and manage automated trading tasks.

The dashboard shows all your tasks with their current status:

| Field | Description |
|---|---|
| Name | Task name |
| Module | Strategy type (Crypto or Limit Bids) |
| Token/Timeframe | e.g. `BTC/5M` |
| Mode | `paper` (simulated) or `live` (real orders) |

Use **Active** / **Paused** filters to sort tasks. Tap **Manage** on any task to view details and controls.

</Warning>
