title: "Portfolio"
description: "Track your open positions and trading performance"
---

Tap **Portfolio** from the main menu to view all your active positions on Polymarket.

Portfolio shows positions for the selected source. Tap a row for details.

## Overview

The overview shows:

| Field | Description |
|---|---|
| Source | Position source: **Manual**, **Auto Trade**, or **Copytrade** |
| Open positions | Number of open positions for the selected source |
| Positions value | Current market value of those positions |
| PnL | Current profit/loss for the selected source |

If there are no positions for the selected source, the page shows an empty state such as **No open Manual positions**.

## Filters

Use the buttons below the portfolio summary to switch views:

| Button | Description |
|---|---|
| Manual | Positions opened manually through Tyche |
| Auto Trade | Positions opened by Auto Trade tasks |
| Copytrade | Positions opened by copytrade subscriptions |
| Open | Show only open positions |
| Closed | Show only closed positions |
| All | Show all positions |
| Refresh | Reload the latest portfolio data |

Use **Back** to return to the previous screen, or **Home** to return to the main menu.

## Open positions

Each open position shows:

| Field | Description |
|---|---|
| Market | Polymarket market question |
| Outcome | Position side, such as YES/NO or Up/Down |
| Shares | Number of shares held |
| Avg → Now | Average entry price and current market price |
| Value | Current position value |
| PnL | Current profit/loss in USD and percent |

Tap **Sell** on an open position to start closing it.

## Pagination

Portfolio results are paginated. The current page is shown at the bottom, for example **Page 1/1**.
