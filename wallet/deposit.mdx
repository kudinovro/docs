---
title: "Deposit"
description: "Fund your Tyche wallet from any supported blockchain network"
---

Tap **Wallet** → **Deposit** to add funds to your wallet.

All deposits are credited as **pUSD** on Polymarket.

<Note>
  Minimum deposit: **$2.00**
</Note>

## Choose a network

Select the network you want to send funds from:

| Network | Type |
|---|---|
| Direct (Polygon) | Send directly on Polygon — no bridging needed |
| Polygon | Send any supported token on Polygon |
| Ethereum | Bridge from Ethereum mainnet |
| Base | Bridge from Base |
| Arbitrum | Bridge from Arbitrum |
| Solana | Bridge from Solana |

## Supported tokens

When depositing via Polygon, you can send any of these tokens — they will be automatically credited as pUSD on Polymarket:

`WBTC` `pUSD` `WETH` `DAI` `SAND` `POL` `USDT`

## How to deposit

<Steps>
  <Step title="Open Wallet">
    From the main menu tap **Wallet**, then tap **Deposit**
  </Step>
  <Step title="Choose network">
    Select the chain you want to deposit from
  </Step>
  <Step title="Send funds">
    Copy the deposit address shown and send your tokens to it
  </Step>
  <Step title="Wait for confirmation">
    Funds will appear in your pUSD Balance after the transaction is confirmed and bridged
  </Step>
</Steps>

<Tip>
  Use **Direct (Polygon)** for the fastest deposits with no bridging fee.
</Tip>
