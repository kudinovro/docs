Tap **Markets** from the main menu to browse Polymarket markets.

Markets is the home for market discovery. Choose a category, type a search query such as **Trump** or **Politics**, or paste a Polymarket market URL.

Open a market row to view details, quick buy, or create a limit order.

## Search

You can find markets in three ways:

| Method | Description |
|---|---|
| Category | Tap a category button to browse matching Polymarket markets |
| Text search | Send a keyword or phrase, such as **Trump**, **Politics**, or **Crypto** |
| Market URL | Paste a Polymarket market URL to open that market directly |

## Available actions

| Action | Description |
|---|---|
| Category button | Open a market feed for that category |
| Search message | Search markets by typed query |
| Polymarket URL | Open the pasted market directly |
| Buy YES / Buy NO | Place a market order from a market detail view |
| Set Limit Order | Create a limit order for a specific market |
| Back | Return to the previous screen |
| Home | Return to the main menu |
